# Novadoc

**A fast, minimal desktop PDF reader.**
Built with Python. Designed to evolve.

🌐 [https://novadoc.io](https://novadoc.io)

---

## What is Novadoc?

Novadoc is a lightweight desktop PDF reader focused on performance and clarity.

No clutter.
No unnecessary features.
Just smooth document viewing — done right.

This is the foundation of a larger vision: a modern document workspace.

---

## MVP Features

* Open and render PDF files
* Smooth scrolling
* Page navigation
* Zoom in / Zoom out
* Clean, modern interface

---

## Built With

* Python
* CustomTkinter
* PyMuPDF
